# üê¶ Birds of Guatemala ‚Äì Android App
An Android application that displays real recordings of birds from Guatemala using the Xeno-Canto API.  
Users can browse bird species, view scientific names, location found, and play real bird calls.

---

## üì≤ How to Download & Compile

### 1) Clone the Repository
```bash
git clone https://github.com/USERNAME/BirdsOfGuatemala.git
```
2) Open with Android Studio

    Launch Android Studio

    Click Open an Existing Project

    Select the cloned folder BirdsOfGuatemala

3) Sync & Build

    Wait for Gradle to finish syncing

    Connect an Android device or emulator

    Click Run 

4) How to Use

    The app will start on the Home Screen. After clicking the "GO" button, you will be able to :

    - Read Guatemalan bird species list

    - Tap a card to hear the real bird sound

    - Tap again to stop playback

    - Double tap on new sound auto-stops previous one and starts new one
